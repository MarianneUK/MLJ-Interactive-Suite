# MLJ Interactive Learning Suite

An interactive collection of English learning games and teaching tools created for **My Language Journey (MLJ)**.

## Vision

The aim of this project is to build a modern, engaging platform where English learners can practise vocabulary, grammar, speaking and British culture through interactive activities.

## Planned Features

### Vocabulary
- Crossword Generator
- Word Search
- Hangman
- Memory Cards
- Word Scramble

### Grammar
- Verb Challenge
- Sentence Builder
- Articles Practice
- Tenses Challenge

### Speaking
- Conversation Cards
- Discussion Questions
- Debate Generator

### British Culture
- Cambridge Quiz
- British Measurements
- British Expressions

### Teacher Tools
- Printable Worksheets
- Vocabulary Lists
- Crossword Generator
- Classroom Activity Generator
